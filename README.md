## 🚀 快速開始

### 環境要求
- [Node.js](https://nodejs.org/) (建議 v18 以上)
- npm 或 yarn

### 安裝步驟

1. **克隆或下載本專案**
2. **安裝依賴套件**
   ```bash
   npm install
   ```
3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```
   伺服器啟動後，可在瀏覽器訪問 `http://localhost:5173/` 查看效果。

### 專案建置
```bash
npm run build
```
建置完成後，靜態檔案將輸出至 `dist/` 目錄，可直接部署於 GitHub Pages, Vercel 或個人伺服器。
